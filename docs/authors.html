Adding New Authors

To add a new author to this network publication list:

  1. Ensure they have an ORCID profile with their publications listed
  2. Add their ORCID iD to the orcid_ids vector in R/fetch_orcid_publications.R
  3. The publication list will update automatically on the next scheduled run
